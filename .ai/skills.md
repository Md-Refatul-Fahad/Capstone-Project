# AI Skills Required

The AI implementing this project must have the following skills.

---

# Skill 1 — Angular Component Generation

AI should generate:

standalone component

template

styles

test file

Example:

product-card.component
product-card.component.html
product-card.component.spec.ts

---

# Skill 2 — Angular Service Creation

AI should generate:

REST style services

typed responses

error handling

Observable responses

---

# Skill 3 — RxJS Usage

AI must know when to use:

switchMap
map
combineLatest
debounceTime

Use RxJS only when necessary.

---

# Skill 4 — Signals State Management

Use Angular signals for:

UI state
cart state
derived values

Example:
cartItems = signal<CartItem[]>([])
total = computed(...)


---

# Skill 5 — Form Handling

Use Reactive Forms.

Include validation.

---

# Skill 6 — Unit Testing

AI should generate:

service tests
component tests
mock API calls

Testing stack:

Vitest
Angular Testing Library

---

# Skill 7 — End to End Testing

AI should generate Playwright tests.

Scenarios must simulate real user flow.
